Зачастую, одна из претензий к Maven - выбор многословного XML для
описания проекта. Polyglot for Maven должен был решить эту проблему,
но проект не "взлетел". Для буквального сокрытия этой проблемы
существует IDEA-плагин Laconic POM.Плагин сворачивает многословные
XML-структуры превращая их в однострочники.Результат работы
плагина.Исходный код плагина доступен на GitHub.    Только
