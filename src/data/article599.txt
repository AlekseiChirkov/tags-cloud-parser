Название: 3CX V18 Release: соединяем бизнес и клиентов – быстро, просто,
эффективно
Теги: Блог компании АйПиМатика, Системное администрирование *, IT-
инфраструктура *
Текст:
Компания 3CX представляет финальную версию 3CX V18 Release, которая
характеризуется лучшим в своем классе качеством связи, отличной
производительностью и надежностью. Коммуникации с клиентом — основной
приоритет любого бизнеса, поэтому V18 сочетает в себе интегрированные
решения как для внутренних коммуникаций, так и для внешнего общения —
в чате, голосом и по видеосвязи.   Новый клиент 3CX и приложение
Windows Desktop App      V18 предлагает пользователям не только
улучшения веб-клиента, но и возможность пользоваться им в виде
нативного приложения Windows Desktop App. Теперь пользователям не
нужно устанавливать расширение 3CX и использовать браузер. Новое
приложение работает с помощью фреймворка Electron, который также
используется в клиентах Microsoft Teams и WhatsApp. Windows Desktop
App отлично интегрируется с операционной системой, обеспечивая быструю
и надежную работу и позволяя работать с вызовами прямо из текущего
приложения, например, CRM-системы, не переключаясь в другое окно.
Улучшенные мобильные приложения для комфортной удаленной работы
интеллектуальное управление кодеками: автоматический выбор кодека в
зависимости от типа сети (Wi-Fi/3G/4G); ускоренное, надежное
переподключение при смене сети или обрыве связи; прямое сотрудничество
с Google и Apple повышает надежность PUSH-уведомлений, позволяя не
терять звонки; добавлена поддержка Bluetooth-гарнитур и автомобильных
устройств громкой связи в мобильных приложениях; интеграция с Siri:
пользователи iOS теперь могут попросить Siri сделать звонок через 3CX.
Поддержка входа Single Sign On (SSO)   Теперь вход в веб-клиент можно
сделать, используя учетную запись Active Directory, Microsoft 365 или
Google. Локальные учетные записи Active Directory поддерживаются через
бесплатную службу интеграции Microsoft Azure AD Connect.   Интеграция
с Microsoft Teams      Интеграция с Microsoft Teams предлагает
доступную альтернативу недешевым тарифным планам Microsoft 365.
Прокси-сервер 3CX для Microsoft Teams обеспечивает надежные
унифицированные коммуникации как для пользователей 3CX, связывающихся
с пользователями Microsoft Teams, так и в обратном направлении.
Пользователи Microsoft Teams могут звонить через SIP-транки 3CX по
местным тарифам, предлагаемым локальными SIP-провайдерами. Компания не
только сократит расходы на связь, но и получит более функциональную
IP-АТС.   Новый голосовой движок: производительность и безопасность
Сервер V18 был существенно переработан для улучшения качества звука,
повышения стабильности при переподключении и усиления безопасности:
класс безопасности A+. Сервер V18 обновлен до Debian 10, последнего
ядра .NET, а OpenSSL и Postgres получили новейшие методы шифрования.
3CX получила сертификацию безопасности A+ в лаборатории Qualys и A в
Immunilabs, что, безусловно, выделяет это решение среди конкурентов.
улучшенное управление кодеками. Мобильные приложения и веб-клиент
теперь используют протокол G711 для уменьшения транскодирования,
снижая нагрузку на процессор и повышая качество звука. Приложения
используют кодек G.711 в сетях Wi-Fi и G.729 в мобильных сетях.
надежное шифрование вызовов благодаря простой настройке SRTP. снижение
нагрузки на процессор и память более чем на 50%. туннель 3CX теперь
объединен с медиасервером для более быстрого восстановления связи
после сбоев сети.   Установка V18 Release   V18 Release доступна для
Linux (устанавливается с помощью 3CX ISO) или как облачный экземпляр
на поддерживаемом облачном провайдере. Также возможна установка на
устройства Raspberry Pi4. Версия 3CX для Windows будет выпущена в
ближайшее время. Если система размещена на хостинге 3CX, вендор
автоматически обновит ее до v18. Систему можно также обновить из
интерфейса управления 3CX.   Обновление сервера:   зайдите в интерфейс
управления 3CX; для безопасности создайте резервную копию конфигурации
в разделе Backup and Restore; выберите Updates; на Debian выберите V18
& OS Upgrade Available Debian 10; на Raspberry Pi4 выберите Major
Release; подтвердите начало обновления; когда ОС и IP-АТС будут
обновлены, вы получите подтверждающий e-mail на адрес администратора.
Замечания к релизу   Для пользователей, использующих ТфОП шлюзы,
Spitfire (UK) и VoIP-провайдеры Connecting Project (IT), опция
Recording External Calls работать не будет. Для этих пользователей
будет выпущено исправление.   Отключена устаревшая опция SMTP для
провайдеров Office365, Outlook и Gmail. Они будут называться Custom
SMTP.   Обновлено приветственное письмо 3CX. Если используется
собственный шаблон письма, его следует также обновить.   Если
используются бриджи 3CX (межстанционные транки), обе системы должны
быть V18.   Некоторые шаблоны CRM убраны из дистрибутива, но могут
быть загружены отдельно с сайта 3CX: 1C, Freshsales, Nutshell,
Opencnam/Everyone API, amoCRM.   После успешного обновления на V18
необходимо перейти в Интерфейс управления -> Параметры -> Системные
сообщения и установить все доступные обновления для языков.
