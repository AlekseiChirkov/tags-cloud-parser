Новости/Открытие кода и данных    OpenNET пишет: «Facebook представил
новый открытый статический анализатор Mariana Trench, нацеленный на

на языке Java. Предоставляется возможность анализа проектов без

секунд), что позволяет применять Mariana Trench для проверки всех
предлагаемых изменений по мере их поступления. Код проекта написан на
языке C++ и распространяется под лицензией MIT». Подробности:

CodeT5 vs Github Copilot: сравнение интеллектуальных генераторов кода
Категория: Статьи/AI & Data Science    Analytics India Mag опубликовал
сравнительный обзор инструментов-помощников в написании кода, вокруг
которых последнее время очень много дискуссий: «Разработка больших
объёмов кода иногда может быть трудоёмкой и утомительной задачей.
Сегодня разработчики ищут методы и инструменты, которые могут помочь
при написании кода и сократить время выполнения работ и повысить
точность для повышения производительности разработки программного
обеспечения. В результате этого открываются возможности автоматической
генерации кода, которые могут развиваться в рамках языков
программирования и IDE, работающих во время компиляции. Автоматическая
генерация кода может выступать в качестве замечательного инструмента с
потенциальными вариантами использования для корпоративных настроек». В
этой статье будут обсуждаться два из самых последних разработанных
инструментов для автоматической генерации кода: открытый Salesforce
CodeT5 и проприетарный Github Copilot. Для каждого инструмента
разобраны:  Архитектура. Функциональность. «Риски» использования.
